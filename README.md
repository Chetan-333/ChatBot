This project is an AI-powered resume chatbot built using LLMs to answer questions based on resume content.

Features:

Retrieval-Augmented Generation (RAG)
Streaming responses for real-time interaction
LangGraph-based workflow design
LangSmith integration for observability

Tech Stack:
LangChain, LangGraph, Streamlit, LangSmith

Status:
🚧 Currently under development – core pipeline implemented, UI and optimization in progress
